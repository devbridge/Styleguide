Installation

                    
We choose Node.js and developed the tool as npm package, because node packages can be easily
                        integrated with existing front-end workflows. Also integration with Gulp is essential if we want
                        to include styleguide automatization in front-end developers workflow.
                    

                


                
    
                    
  • 
                        Install local package to your project directory
                        
    
                            npm install devbridge-styleguide --save-dev
                            |
                        
    
                    
    • 
                    
  • 
                        Create Gulp task

    var styleguide = require('devbridge-styleguide');
gulp.task('start-styleguide', function () {
  styleguide.startServer();
});
    
                    
    • 
                    
  • 
                        Start styleguide server 
    
                        
    
                            gulp start-styleguide
